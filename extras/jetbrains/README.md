# Prairie Theme for JetBrains IDEs

A complete light, warm theme for JetBrains IDEs (IntelliJ IDEA, PyCharm, WebStorm, etc.).

## Installation Options

### Option 1: Plugin Installation (Recommended)
The theme is packaged as a proper JetBrains plugin for easy installation:

1. Navigate to the `prairie-theme-plugin/` directory
2. Follow the build and installation instructions in that folder's README
3. Install the generated plugin ZIP file via **Settings** → **Plugins** → **Install Plugin from Disk**

### Option 2: Manual File Import (Legacy)
If you prefer to import the theme files manually:

#### Editor Color Scheme
1. Go to **Settings** → **Editor** → **Color Scheme**
2. Click the gear icon → **Import Scheme**
3. Select `prairie.icls` from this folder
4. Apply the "Prairie" color scheme

#### UI Theme (Optional - for complete theming)
1. Go to **Settings** → **Appearance & Behavior** → **Appearance**
2. Click **Theme** dropdown → **Browse Themes...**
3. Click the gear icon → **Import Theme**
4. Select `prairie.theme.json` from this folder
5. Apply the "Prairie" UI theme

## Features

- Light background with warm prairie tones
- Complete IDE theming (editor + interface)
- Optimized for readability with high contrast
- Git integration colors for file status
- Anonymous Pro font at 16pt
- Consistent color palette throughout IDE

## Files

- `prairie.icls` - Editor color scheme (standalone)
- `prairie.theme.json` - UI theme (standalone)  
- `prairie-theme-plugin/` - Complete plugin package (recommended)
